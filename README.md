# Soluções do TOP(The Odin Project)

Este repositório contém as soluções de exercícios práticos durante meus estudos de **Css** e **JavaScript** no TOP, com foco em design responsivo, lógica de programação e versionamento com Git/GitHub.

Os exercícios foram realizados como parte do meu aprendizado contínuo em desenvolvimento web Full-Stack.

---

## 📂 Estrutura do Repositório

Cada exercício está organizado em sua própria pasta, facilitando a leitura, manutenção e controle de versão.

- Html5 & Css3

- Visual Studio Code
